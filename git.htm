Setting Up the Environment

    Install Git on Windows

    1. Go to the official Git website.
    2. On the homepage, you will find a "Download" button. Click on it.
    3. You will be directed to a page where you can choose your operating system after clicking the download button.
    4. Once the download is complete, locate the downloaded file and double-click it to run the isntaller.
    5. After the installation is complete, you can verify that Git was installed correctly by opening a terminal and typing "git --version".

    Install VS Code on Windows

    1. Go to the official Visual Studio Code website.
    2. Click on the "Download for Windows" button.
    3. Once the download is complete, run the installer (VSCodeUserSetup-{version}.exe).
    4. By default, VS Code is installed under C:\Users\{Username}\AppData\Local\Programs\Microsoft VS Code.

    Configure Git within VS Code (user name, email)

    1. Open VS Code on your computer.
    2. Go to the View menu and select "Terminal" to open the integrated terminal.
    3. Before configuring Git, ensure that Git is installed on your system by typing 'git --version' in the terminal and pressing Enter.
    4. In the terminal type "git config --global user.name "Your Name" to set your Git username. Then press Enter.
    5. Type "git config --global user.email "your.email@example.com" to set your Git email address and press Enter.
    6. You can verify that your username and email are configured correctly by typing "git config --global --list" in the Terminal and press Enter. Your username and email should pop up.

Creating and Cloning a GitHub Repository

    Creating a new repository on GitHub

    1. Create a new repository on GitHub by navigating to your GitHub account and clicking on the "New" button.
    2. Fill in the repository name, description, and choose other settings as needed.
    3. Click on the "Create repository" button.
    4. Once your repository is created, copy the repository URL.

    Cloning the repository using VS Code's integrated Git support

    1. In VS Code, open the command palette by pressing Ctrl + Shift + P.
    2. Type "Git: Clone" and select the option "Git: Clone" from the dropdown.
    3. Paste the repository URL and choose the directory where you want to clone the repository.
    4. Press Enter to clone the repository.

Basic Git Operations in VS Code

    Performing basic Git operations:

    1. Open the Source Control view by clicking on the Git icon in the Activity Bar on the side.
    2. Review the changes in the Changes section.
    3. Stage changes by clicking the "+" button next to each changed file.
    4. Enter a commit message in the input box at the top of the Changes section and press Enter to commit.
    5. Click the ... menu in the top-right corner of the Source Control view and select "Push" to push your commits to the remote repository.
    6. To pull changes, click the ... menu in the top-right corner of the Source Control view and select "Pull".


Branching with VS Code

    Using VS Code to create and switch between branches

    1. To create a new branch, open the Source Control view by clicking on the Git icon in the Activity Bar on the side.
    2. Click on the "Branches" button at the top of the Source Control view to display a list of branches.
    3. Click on the "+" button next to "Local Branches" to create a new branch.
    4. Enter a name for your new branch and press Enter.
    5. To switch between branches, click on the branch name in the status bar at the bottom-left corner of the window. Select the branch you want to switch to from the dropdown list.

Working with Remote Repositories

    Adding and Managing Remote Repositories

    1. Open your file in VS Code.
    2. Access the Source Control view.
    3. Clink on ... menu and select "Add Remote...".
    4. Enter the remote repository URL and details.
    5. Press Enter to add the remote repository.

    Pushing and Pulling

    1. In the Source Control view, click on the ... and select "Push" to push changes to the remote.
    2. Click on the ... menu and select "Pull" to pull changes from the remote.
    3. Go back to your explorer tab and you will see the remote Github file.

Pull Requests and Code Reviews within VS Code

    Creating Pull Requests

    1. Make changes to your code.
    2. Commit your changes and push them to a branch on GitHub.
    3. In VS Code, click the GitHub icon in the Activity Bar to open the GitHub pane.
    4. Click on "Create Pull Request" and follow the prompts to create the pull request.

    Reviewing Pull Requests

    1. Go back to GitHub.
    2. Open the Pull Requests tab and select the pull request you want to review.
    3. Review the changes made in the pull request.
